# Crypto Tracker App 📱

A professional Flutter application for tracking cryptocurrency prices, news, and maintaining a watchlist.

## Features

✅ **3 Live APIs**
- CoinGecko API (Trending cryptos + Prices)
- NewsAPI (Latest crypto news)
- Real-time price updates (USD & INR)

✅ **Search Functionality**
- Search any cryptocurrency dynamically
- Instant results with live prices

✅ **Watchlist**
- Save favorite cryptos locally
- Data persists after app closes
- Remove from watchlist anytime

✅ **Detailed Views**
- Tap any crypto → See full details
- USD & INR prices
- Add/remove from watchlist

## Technologies Used

- **Flutter** - UI Framework
- **Dart** - Language
- **HTTP Package** - API calls
- **SharedPreferences** - Local storage

## Getting Started

1. Clone the repo
2. Run `flutter pub get`
3. Run `flutter run`

## API Keys

- CoinGecko: Free (no key needed)
- NewsAPI: Get free key from https://newsapi.org/
